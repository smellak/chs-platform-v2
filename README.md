# Aleph Platform

Open-source platform for managing internal business applications with unified identity, access control, and AI agent orchestration.

## Quick Start

```bash
git clone https://github.com/smellak/aleph-platform.git
cd aleph-platform/docker
cp .env.example .env
docker compose up
```

Open http://localhost:3000 — Login: admin / admin123

## Stack

- Next.js 15 (App Router)
- PostgreSQL + Drizzle ORM
- Tailwind CSS
- TypeScript (strict)
